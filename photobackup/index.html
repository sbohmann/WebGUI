photobackup server

Install docker.

Run the script run_image.sh

Syntax:

./run_image.sh <storage directory>

Inside your storage directory, the directory tree backup/photobackup-server/... will be created and used for storing backups, so please make sure that that does not conflict with existing data.

Viewing backed up photos and videos:

<host name or address>:8080/gallery
